Checkpoint 1
Была реализована внутренняя логика игры.

GameSession - объект, хранит текущее состояние игры. После старта игры только один игрок может изменить состояние своим ходом. Поэтому этот класс теоретически (быть может правильно добавив ему многопоточность) можно использовать для многопользовательской игры.

Ship - объект, который хранит информацию о корабле и определяет его поведение. Был добавлен паттерн Factory на тот случай, если захочется добавить нестандартные корабли в игру.

Command - реализует соответствующий паттерн.

При желании SayCommand можно определить как Observer. Для этого достаточно изменить реализацию.

Как играть

Зарегистрировавшись и начав новую игру, пользователь попадает в меню выбора кораблей. 
Игрок выбирает номер одного из свободных кораблей. Затем вводит координаты двух клеток - начала и конца корабля. Ввод в формате строки из двух символов {A..J,0..9}.
Чтобы убрать корабль с поля, нажмите -1 и затем введите координату любой из клеток, покрытых лишим судном.

Для атаки выбор клетки тот же.
